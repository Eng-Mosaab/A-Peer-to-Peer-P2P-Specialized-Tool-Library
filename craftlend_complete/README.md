
## Features included
- User Role Management Module
- User Manipulation Module
- Controlling Resources Module
- Reservation and Rescheduling Module
- Generating Reports Module
- Sending Emails or Notifications Module
- File Uploaders
- Authentication
- Password Hashing
- Role-based Authorization
- PHP Sessions
- CRUD Operations
- AJAX Scenario
- Database Implementation
- Front-End Pages Implementation
- MVC Structure + OOP + Database Connection

## Run steps
1. Copy the folder into `C:\xampp\htdocs\craftlend_complete`
2. Import `database/schema.sql`
3. Open `http://localhost/craftlend_complete/public/index.php`
4. Register a new account and choose a role

## Notes
- No demo users or demo tools are included.
- Emails are logged inside the system as `email_logs` records.
- Reports can be viewed and exported as CSV.
